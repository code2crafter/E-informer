📖 E-Informer - Documentation

👋 Introduction

E-Informer is a web-based PCB image analyzer that uses YOLOv8 object detection models to identify and label components on PCB images for:

  • 🔌 Arduino UNO
  • 🍓 Raspberry Pi
  • 🧩 Custom PCB Boards

📂 Uploading Images

Navigate to the Choose Board page and select your board type. Then:

  1. Click on the upload form.
  2. Select an image file (.jpg, .png).
  3. Click Upload Image.
  4. Wait for the result — the detected components will appear below.

🖼️ Supported Image Types

Only common image formats like .jpg, .jpeg, and .png are supported. Make sure the image is clear and taken from top view of the PCB.

📡 Mobile Access

To access this site from your mobile phone:

  • Connect your phone and PC to the same Wi-Fi network.
  • Run the app using app.run(host="0.0.0.0").
  • Visit http://your_pc_ip:5000 from your mobile browser.
